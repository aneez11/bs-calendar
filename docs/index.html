Getting Started

Installation

# Install the package
npm install bs-calendar

# With React components (optional peer dep)
npm install bs-calendar react
yarn add bs-calendar
pnpm add bs-calendar

Basic Usage

import { toAD, toBS, toBSString, daysInMonth, todayBS } from 'bs-calendar'

// BS → AD
const adDate = toAD(2080, 1, 1)
// → Date('2023-04-14')  — Nepali New Year

// AD → BS
const bs = toBS(new Date())
// → { year: 2080, month: 1, day: 14 }

// Format
toBSString(new Date())           // "2080-01-14"
todayBS()                         // { year, month, day }
daysInMonth(2080, 1)            // 30
const { toAD, toBS, daysInMonth } = require('bs-calendar')

const ad = toAD(2080, 1, 1)
console.log(ad) // Date('2023-04-14')

Nepali (Devanagari) Formatting

import { toDevanagariNumeral, toFormattedNepaliBS, nepaliMonthName } from 'bs-calendar'

toDevanagariNumeral(2080)       // "२०८०"
nepaliMonthName(1)              // "वैशाख"
toFormattedNepaliBS(new Date('2023-04-14'))
// → "२०८० वैशाख १"

Calendar Grids

Generate 42-cell grids (6×7, Sun–Sat) for UI rendering. Each cell has both BS and AD dates.

import { getMonthGrid, getADMonthGrid } from 'bs-calendar'

// BS grid — 42 CalendarCell objects
const bsCells = getMonthGrid(2080, 1)
// { bsDay, adDate, isOtherMonth, isToday, ... }

// AD grid — 42 ADCalendarCell objects
const adCells = getADMonthGrid(2024, 0)
Components

BSCalendar

Interactive calendar component with Tailwind, Bootstrap & default themes. Supports events, holidays, and full customization.

Minimal

import { BSCalendar } from 'bs-calendar/react'

<BSCalendar
  view="both"
  language="both"
  theme="default"
  showNavigation
  onDateSelect={(ad, bs) => console.log(bs)}
/>

With Events & Holidays

import { BSCalendar } from 'bs-calendar/react'
import type { CalendarEvent, Holiday } from 'bs-calendar/react'

const events: CalendarEvent[] = [
  { id: '1', title: 'Meeting',  date: '2024-01-15', time: '10:00', color: '#2563eb' },
  { id: '2', title: 'Festival', date: '2080-10-15', dateType: 'bs', color: '#f59e0b' },
]

const holidays: Holiday[] = [
  { date: '2080-10-15', dateType: 'bs', name: 'Maghe Sankranti', nameNP: 'माघे सङ्क्रान्ति' },
]

<BSCalendar
  theme="tailwind"
  events={events}
  holidays={holidays}
  showEventList showHolidayLabels
  onEventClick={(ev) => alert(ev.title)}
/>

BSDatePicker

Date input with interactive calendar dropdown. Supports AD and BS modes with auto-conversion between them.

import { BSDatePicker } from 'bs-calendar/date-picker'

<BSDatePicker
  mode="both"
  theme="default"
  placeholder="Select date..."
  onChange={(ad, bs) => console.log(ad, bs)}
/>
<BSDatePicker
  mode="both"
  theme="tailwind"
  {/* Type "2080-01-01" → auto-converts to AD 2023-04-14 */}
  {/* Type "2023-04-14" → auto-converts to BS 2080-01-01 */}
  onChange={(adDate, bs) => {
    console.log('AD:', adDate.toISOString())
    console.log('BS:', bs)
  }}
/>

Guide

Date Formatting

Use formatBSDate and formatADDate for custom date output in any format.

import { formatBSDate, formatADDate } from 'bs-calendar'

formatBSDate(new Date(), 'YYYY-MM-DD')       // "2080-01-15"
formatBSDate(new Date(), 'DD MMMM YYYY')      // "15 Baisakh 2080"
formatBSDate(new Date(), 'YYYY MMMM-NP DD')   // "2080 वैशाख 01"

formatADDate(new Date(), 'DD/MM/YYYY')       // "15/01/2024"

Styling

Three ways to customize appearance — from per-element inline styles to full custom cell rendering.

// Default theme — override per-element styles
<BSCalendar
  styles={{
    cell: { background: '#1e293b', borderRadius: 6 },
    cellToday: { background: '#1e3a5f', border: '2px solid #3b82f6' },
    bsNumber: { color: '#f1f5f9', fontSize: 14 },
    adNumber: { color: '#64748b', fontSize: 10 },
    cellOtherMonth: { opacity: 0.15 },
    navButton: { background: '#334155', border: 'none' },
  }}
/>
<BSCalendar
  theme="tailwind"
  classNames={{
    cell: 'bg-slate-800 hover:bg-slate-700 rounded-lg',
    cellToday: 'bg-blue-900 ring-2 ring-blue-500 font-bold',

    adNumber: 'text-slate-500 text-xs',
    navButton: 'px-3 py-1 rounded-lg bg-slate-700 hover:bg-slate-600 border-0 text-white',
  }}
/>

  theme="bootstrap"
  classNames={{
    cellToday: 'bg-info bg-opacity-25 fw-bold border-info',

  }}
/>

Events & Holidays

Both BSCalendar and BSDatePicker accept events and holidays. Use dateType: 'bs' for BS dates.

import type { CalendarEvent, Holiday } from 'bs-calendar/react'

// AD and BS dates are both supported
const events: CalendarEvent[] = [
  { id: '1', title: 'Meeting',       date: '2024-01-15',           color: '#3b82f6' },
  { id: '2', title: 'Nepali New Year', date: '2081-01-01', dateType: 'bs', color: '#22c55e' },
]

const holidays: Holiday[] = [
  { date: '2081-01-01', dateType: 'bs', name: 'New Year', nameNP: 'नयाँ वर्ष' },
]

CalendarEvent fields: id, title, date, dateType?, endDate?, color?, time?, description?, location?, allDay?, type?

Holiday fields: date, dateType?, name, nameNP?, type?, color?
